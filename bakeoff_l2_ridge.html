Key insights from the bakeoff
Ridge beats Lasso on this dataset — here's why
Ridge achieved Test R² of 66.24% vs Lasso's 63.74%. Because most features in the California Housing dataset carry genuine predictive signal, L2 (which shrinks all weights but keeps all features) outperforms L1 (which eliminates features). Lasso is better when truly irrelevant features exist; Ridge is better when all features contribute.
Why XGBoost still scores 83.75% vs Ridge's 66.24%
The ~17.5% gap confirms that California housing prices follow non-linear relationships that Ridge — a linear model — cannot capture. Ridge's 66.24% represents the hard ceiling of linear extraction from this dataset. Non-linear models like XGBoost capture interaction effects between income, location, and density.
The weight decay proof — 91.7% coefficient reduction
From Version 1 (Alpha=0.1) to Version 10 (Alpha=50,000), the average coefficient magnitude shrank by 91.7%. This is a direct empirical demonstration of Weight Decay in neural networks — where the L2 penalty suppresses extreme weights, preventing any single feature from dominating predictions.
Solver convexity — a mathematical guarantee unique to Ridge
Testing 4 solvers (auto, sag, lsqr, cholesky) at the same Alpha produced virtually identical results. This proves Ridge Regression's loss landscape is perfectly convex — unlike neural networks where different random seeds give different local minima, Ridge always converges to the same single global optimum.
Catastrophic underfitting at Alpha = 50,000 (V10)
Version 10 demonstrated the danger of excessive Weight Decay — Test R² collapsed to just 23%. The L2 penalty became so strong it suppressed genuine predictive weights alongside noise, proving that over-regularization destroys model capacity just as much as no regularization causes overfitting.
